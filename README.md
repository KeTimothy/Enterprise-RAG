# Enterprise RAG Assistant

A production-style Retrieval-Augmented Generation (RAG) assistant built with FastAPI, PostgreSQL, pgvector, and local LLMs via Ollama.

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- pgvector
- Docker
- Ollama
- Hugging Face
- Git
