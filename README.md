# Enterprise RAG Assistant

An end-to-end Retrieval-Augmented Generation (RAG) system built with:

- FastAPI
- PostgreSQL
- pgvector
- Docker
- SQLAlchemy
- OpenAI-compatible LLMs (later)
- LangChain (later)

## Current Progress
